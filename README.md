💸 SpendSmart — Student Expense Analyzer

SpendSmart is a modern and interactive expense tracking web app designed specifically for students. It not only tracks your expenses but also provides deep insights, spending patterns, waste analysis, and smart saving suggestions.

🚀 Features
➕ Add & Track Expenses

Add expenses with description, amount, category, and date

Categorized tracking (Food, Transport, Shopping, etc.)

Filter by category and time (week/month/all time)

📊 Dashboard Analytics

Monthly spending summary

Daily average & highest spending day

Smart category insights with budget comparison

📈 Spending Patterns

Daily spending trend (last 30 days)

Weekly spending analysis

Day-wise spending behavior

Category heatmap visualization

🔥 Waste Detection

Detects overspending based on thresholds

Highlights money-wasting categories

Provides actionable tips to reduce expenses

💰 Savings Suggestions

Personalized saving tips

Potential monthly savings estimation

Savings score based on spending habits

🛠️ Tech Stack

HTML5

CSS3 (Modern UI with custom design)

JavaScript (Vanilla JS)

Chart.js (for data visualization)

LocalStorage (for saving user data)
